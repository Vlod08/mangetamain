Compute and save the cosine similarity matrix based on time-related tags.

Parameters:

  • anchor (Path | str) – Typically Path(__file__) or the project root (used by RecipesDataset).

  • out_path (Path | str, default 'data/processed/time_similarity.parquet') – Where to save the similarity matrix.

  • sample_n (int | None) – If provided, randomly sample this many recipes before computing similarity.

  • random_state (int, default 1) – Random seed for reproducible sampling.

  • as_csv (bool, default False) – If True, saves the matrix as CSV instead of Parquet.

Returns:

Path to the saved similarity file.
